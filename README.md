# Cushion

Cushion is a personal financial planning app for the period between one salary and the next.

Its main purpose is to answer one practical question:

> can i afford this purchase before my next salary?

The MVP is intentionally focused on planning rather than accounting. It helps the user reserve money for known obligations, plan everyday spending and understand how much money remains available.

## MVP documentation

- `docs/product-principles.md` — product principles and non-goals
- `docs/product-spec.md` — complete MVP requirements
- `AGENTS.md` — implementation instructions for Codex

These documents are the source of truth for the MVP.

## Local development

Requirements: Node.js 22 or newer.

```bash
npm install
npm run dev
```

Checks:

```bash
npm test
npm run build
```
